# **Useful Scripts for PDF Privacy and Cleanup**

This repository provides a simple yet powerful Python script to help you **sanitize PDF files offline**, ensuring your privacy and compliance with upload requirements on various platforms.

---

## **Why Use This Script?**

Many websites reject PDFs that contain:

- **Encryption or password protection**
- **Multiple layers**
- **Digital signatures**
- **Security headers**

This script removes all of these elements and generates a clean, upload-ready PDF—**without sending your sensitive documents to any online service**.

---

## **Benefits**

- **Privacy First**: Everything runs locally on your machine. No data leaves your system.
- **Universal Compatibility**: Works with documents like:
    - Bank statements
    - Payslips
    - Agreements
    - Any PDF with security restrictions
- **Quick and Reliable**: One command, and your PDF is ready for upload.

---

## **Prerequisites**

Before running the script, install the required library:

## **Features**

- Remove **encryption** from PDFs
- Strip **layering** and flatten the file
- Eliminate **digital signatures**
- Remove **security headers**

---

## **Result**

You’ll get a **new, clean PDF** that:

- Is free from security restrictions
- Can be uploaded to websites that reject protected or layered PDFs
- Maintains your privacy by staying offline

---

## **Why This Matters**

Uploading sensitive documents to online converters can compromise your data. This script offers the **best method for privacy-conscious users** who prefer offline solutions.
