# useful-scripts
Useful scripts to automate some of the useful tasks and do stuff offline if you don't want to give your data to the internet

Tips before running the script.
run the command to install the library PyPDF2
- pip install PyPDF2

## This python script is for the following purposes:
- To remove encryption from the pdf
- To remove the layering in the pdf
- To remove the digital signatures in the file
- To remove security headers in the pdf

### Result:
- A new pdf is generated which can be used to upload on website, which do not accept the pdf's with digital security headers or multiple layered pdf's.
- Secure as it is offline and no data will be uploaded anywhere. Best method if you prefer privacy.
Some document examples - Bank Statements, Payslips, Agreements etc.

Enjoy the offline privacy and not uploading the important pdf documents over the internet.
